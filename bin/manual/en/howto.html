How to compile BonkEnc from source?

In order to build BonkEnc from source you should download the latest version of the BonkEnc Component Development Kit (CDK) from www.bonkenc.org first.

You will also need the latest smooth Class Library sources from www.smooth-project.org and of course the BonkEnc source code package or BonkEnc sources checked out from CVS.

  • extract the contents of bonkenc-cdk-1.x.zip to anywhere you like
  • create a directory structure like this:
    /cdk (contains directory system and cdk.bat)
    /cdk/bonkenc (rename directory from bonkenc-1.1.x.zip package)
    /cdk/smooth (rename directory from smooth-0.8.x.y.zip package)
  • move Makefile.mingw from smooth/makefiles to /cdk/smooth/Makefile
  • run cdk.bat
  • enter: cd smooth && make programs && cd ../bonkenc && make

How to install encoder/decoder xy?

Nero Digital HE-AAC Encoder

To install the Nero Digital HE-AAC Encoder for use with BonkEnc, follow these steps:

  • Download the Nero AAC Codec ZIP package from www.nero.com.
  • Copy the file NeroAACEnc.exe from the ZIP package to the codecs/cmdline subdirectory of your BonkEnc installation path.
Fraunhofer MP3 Surround Encoder

To install the Fraunhofer MP3 Surround Encoder for use with BonkEnc, follow these steps:

  • Download the Win32 command line encoder ZIP package from www.all4mp3.com.
  • Copy the file mp3sEncoder.exe from the ZIP package to the codecs/cmdline subdirectory of your BonkEnc installation path.
Helix DNA Producer RealAudio Encoder

To install the Fraunhofer MP3 Surround Encoder for use with BonkEnc, follow these steps:

  • Create a Helix Community account at helixcommunity.org. You will need this account in order to be able to download the Helix DNA Producer application.
  • Download the Helix DNA Producer Command Line App ZIP package from helix-producer.helixcommunity.org.
  • Create a new directory named helix inside the codecs/cmdline subdirectory of your BonkEnc installation path.
  • Copy the contents of the directory producer from the ZIP package to the newly created codecs/cmdline/helix subdirectory of your BonkEnc installation path. The file producer.exe should now reside inside the helix directory along with several other files and directories from the Helix DNA Producer package.
TiMidity Software MIDI Synthesizer

To install the TiMidity Software MIDI Synthesizer for use with BonkEnc, follow these steps:

  • Download the TiMidity CVS binary ZIP package from sourceforge.net.
  • Download the Freepats ZIP package from freepats.opensrc.org.
  • Create a new directory named timidity inside the codecs/cmdline subdirectory of your BonkEnc installation path.
  • Copy the file timidity.exe from the TiMidity ZIP package to the newly created codecs/cmdline/timidity subdirectory of your BonkEnc installation path.
  • Copy the contents of the directory freepats from the Freepats ZIP package to the newly created codecs/cmdline/timidity subdirectory of your BonkEnc installation path. The file crude.cfg should now reside inside the timidity directory along with several other files and directories from the Freepats package.

  • Download the TAK ZIP package from thbeck.de.

How to enable Windows Media Audio decoding support?

To enable Windows Media Audio (WMA) support in BonkEnc, you need the WMA plug-in from Winamp 5.21 (plug-ins from newer versions of Winamp will not work with BonkEnc). Please follow these steps to install the WMA plug-in for BonkEnc.

  • download and install Winamp 5.21 from www.oldversion.com
  • copy the in_wm.dll file from the Winamp/plugins directory to the BonkEnc/plugins directory

How to add a new translation to BonkEnc?

Please use the smooth Translator utility included in the BonkEnc distribution and open an existing translation file (found in the BonkEnc/lang directory). Then change the language settings to reflect the translation you want to create and start translating the individual text strings. Save your translation as a new file matching the pattern bonkenc_??.xml (replace ?? with your language code), restart BonkEnc and try selecting the new translation in the configuration dialog.

Please submit your translation to translations@bonkenc.org, so it can be included in the official distribution.

How to remove BonkEnc from my computer?

Recent versions of BonkEnc come with an uninstaller that you can execute from the Start menu.

In case you removed the Start menu group or are unable to find it, it is completely sufficient to delete the application directory to uninstall BonkEnc. BonkEnc does neither create any Windows registry entries nor store any active components outside its installation directory.
