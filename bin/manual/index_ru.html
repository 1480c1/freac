Руководство пользователя BonkEnc

Руководство по копированию CD как для опытных, так и для начинающих пользователей.

С расцветом MP3s в наше время каждый хочет иметь их, копируя свои CD в аудио файлы. К сожалению, при использовании многих широкораспространенных утилит качество оставляет желать лучшего. К счастью есть несколько свободных и Open Source программ, которые позволяют получить качественное звучание музыки как для вашего компьютера, так и для iPod.

BonkEnc - это многоцелевая программа для копирования CD, кодирования и конвертирования аудио файлов включая MP3, Ogg Vorbis и Wave форматы. Последнюю версию BonkEnc Вы можете скачать с сайта BonkEnc.

Данное руководство в процессе разработки. Руководство для начинающих уже готово, но есть еще немного работы, которая должна быть выполнена над расширенными уроками.
